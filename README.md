# 🚇 Smart Metro Management System

## 📌 Overview
This is a Java-based desktop application developed using Swing and MySQL.  
It helps manage metro operations such as stations, trains, and admin activities.

---

## ⚙️ Technologies Used
- Java (Swing)
- JDBC
- MySQL
- OOP Concepts

---

## ✨ Features
- Admin Login with SHA-256 Security
- Dashboard Analytics
- Add & Manage Stations
- Train Simulation
- Activity Logging
- Map Integration (OpenStreetMap)

---

## 🚀 How to Run
1. Import project in NetBeans
2. Setup MySQL database using `smartmetro.sql`
3. Run the project
